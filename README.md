# c_sort

C语言实现的任意类型数组排序。

|函数          |描述    |时间复杂度（平均）    |
|:-----------:|:------:|:-------------------:|
|`bubbleSort` |冒泡排序 |`O(n^2)`             |
|`selectSort` |选择排序 |`O(n^2)`             |
|`insertSort` |插入排序 |`O(n^2)`             |
|`shellSort`  |希尔排序 |`O(nlogn)` ~ `O(n^2)`|
|`quickSort`  |快速排序 |`O(nlogn)`           |
|`mergeSort`  |归并排序 |`O(nlogn)`           |
|`heapSort`   |堆排序   |`O(nlogn)`           |
